Deskripsi Pembuatan Website

Author : Gian Y.R

Persiapan

1. Alat

Text Editor: Kalau saya menggunakan Brackets sebagai text editor karena ringan dan memiliki fitur yang mendukung untuk pengembangan.

Browser Web: Saya menggunakan Mozilla Firefox sebagai browser web untuk melihat hasil website dan melakukan debugging.

2. Pemahaman Dasar

HTML: Saya belajar dasar-dasar HTML untuk membangun struktur website.

CSS: saya belajar CSS untuk menata tampilan website, sepreti warna, font, ukuran, tata letak, dan lainnya.

Javascript: saya belajar Javascript untuk menambahkan interaktiitas pada website, seperti animasi, efek, dan validasi formulir.

Langkah-langkah Pembuatan

1. Membuat File HTML

~ Buka Brackets dan buat file baru dengan ektensi .html

2. Menambahkan Konten

~ Tambahkan konten website, seperti teks, gambar, link dan lainnya.

3. Menata Website dengan Css.

~ Buat file CSS dengan Ekstensi .css
~ Tulis kode CSS untuk menata tampilan website, seperti warna, font, ukuran, tata letak, dan lainnya.
~ Hubungkan file CSS ke file HTML

4. Menambahkan Interkativitas dengan Javascript

~ Buat file Javascript baru dengan ekstensi .js
~ Tulis kode Javascript untuk menambahkan interaktivitas pada website, seperti animasi, efek, dan validasi formulir.
~ Hubuungkan file Javascript ke file HTML.

5. Menjalankan Website

~ Simpan file HTML, CSS, Javascript.
~ Buka file HTML di Mozilla Firefox Browser.
~ Website akan ditampilkan di browser.

6. Pengujian dan Debugging

~ Uji website di berbagai browser dan perangkat untuk memastikan kompabilitas.
~ Gunakan alat debugging di Mozilla Firefox untuk menemukan dan memrperbaiki bug atau kesalahan pada kode

Kendala

1. Susah untuk publish website ke internet

dikarenakan domain dan hostingnya berbayar kalaupun ada yang gratis bisanya cuman project kecil karena project saya berukuran lumayan besar jadi susah untuk publishnya.

2. Waktu

Kalau diberi banyak waktu mungkin project/website ini akan jadi lebih interaktif dan selesai/jadi. Saya kekurangan waktu karena ada kesibukan lain jadi project ini cuman bisa selesai sekitar 60%/70%.
